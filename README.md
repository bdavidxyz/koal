# README

Minimalistic Rails 8 app.

- Responsive navbar
- Tailwind 4
- Sign in screen
- Sign up screen
- Debugbar locally
- Custom Google Font
- Cohesive design system
- Pre-built landing page
- Unpoly
- Build-in toast solution both for frontend and backend

## Prerequisites

- Git
- PostGre
- NodeJS >= 2
- Ruby > 3.3

## .env file

Content of .env:

```
DATABASE_URL=postgres://localhost:5432/koal_development

SMTP_SERVER=
SMTP_PORT=
SMTP_USERNAME=
SMTP_PASSWORD=
```
