# commands.md

These are the rules related to how to launch command in the CLI.

## Guidelines

- Launch every "rails" command with "bin/rails"
- Launch rubocop only with "bin/rubocop"
- If you need to start the rails server, use "bin/dev", but be aware it's usually already running when I give you instruction (it runs on localhost:3031)
- If you need to touch JS stuffs, use npm, not yarn
